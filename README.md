# water_puzzle
A fast solver for water sort puzzle.

### Example
Run
```
$ ./water_puzzle ",绿绿黄黄,红红黄黄,绿红蓝蓝,绿红蓝蓝"
```
Output:
```
[1] #2 ――→ #1
[2] #4 ――→ #1
[3] #5 ――→ #1
[4] #3 ――→ #4
[5] #3 ――→ #5
[6] #2 ――→ #3
[7] #4 ――→ #2
[8] #5 ――→ #2
[9] #4 ――→ #5
```
